一个简单的网站，可以自定义目标url和内容关键词，当目标网站出现了指定关键词后会用邮件进行通知。

我曾用它来订阅某些报名公告、分数查询通知，还有证书领取公告，还是挺好用的。

第一次用要修改monitor.php里面的SMTP信息和db_connection.php里面的数据库信息

宝塔计划任务里添加一个定时任务

访问URL-GET[ https://your.domain.com/monitor.php ]

频率10秒到60秒之间，自己看着来
